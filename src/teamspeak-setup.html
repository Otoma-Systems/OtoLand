TeamSpeak Connection Guide

Follow these steps to connect to the Otoma Systems TeamSpeak server.

Step 1: Download TeamSpeak

If you haven't already, download and install the TeamSpeak client from teamspeak.com.

Then register on myteamspeak.com/register.

Screenshot of TeamSpeak download page
(Place image here)

Step 2: Launch TeamSpeak

Open the TeamSpeak application on your computer after installation.

Screenshot of TeamSpeak main window
(Place image here)

Step 3: Open the Connect Dialog

Click on Connections in the top menu bar, then select Connect.

Screenshot highlighting Connections → Connect
(Place image here)

Step 4: Enter Server Address

In the connection dialog, enter the following server address:

ts.otoma.com.br

Set your preferred nickname that will be visible to other users.

Screenshot of the Connect dialog with server & nickname
(Place image here)

Step 5: Add to Bookmarks

Before connecting, click on Bookmarks > Add to Bookmarks to save this server for quick access in the future.

Screenshot highlighting Bookmarks → Add to Bookmarks
(Place image here)

Step 6: Connect

Click Connect to join the Otoma Systems TeamSpeak server. You should now be connected to the voice channels.
